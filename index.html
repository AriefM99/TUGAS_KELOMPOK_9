Slide Kekurangan Dari React Js

  1. Dokumentasi resmi React kurang tertata dengan baik dibandingkan dengan kebiasaan dokumentasi framework lainnya. Tetapi dokumentasi yang disediakan di luar dokumentasi resmi sudah banyak.
  2. React fokus pada bagian View dari suatu aplikasi. Jika ingin membangun aplikasi dalam skala besar, kita harus menyertakan beberapa library luar untuk menangani bagian lain dari suatu aplikasi.
  3. Untuk membuat routing pada React, kita harus menyertakan file dari luar.
